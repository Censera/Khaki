Small program for creating folder project for C programming language.

## Usage

### New project

To create a new project use the action `new` followed by the name of the project

``` shell
khaki new my_project
```

### Run project

Use the action `run` followed by the by excutable name and `options` as the compiler flags

```shell
khaki run project -O3
```

### Show

For showing listing the folder content

```shell
khaki show -L 2
```

### Config 

```shell
khaki config
```
